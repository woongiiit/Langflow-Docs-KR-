Sequential tasks agent

여러 Agent 컴포넌트를 활용한 멀티 에이전트 애플리케이션을 위해 Sequential Tasks Agent 플로우를 구축하세요.

각 에이전트는 LLM 모델과 고유한 도구 세트를 가지고 있으며, Prompt 컴포넌트가 Agent Instructions 필드에 연결되어 에이전트의 동작을 제어합니다. 예를 들어, Researcher Agent는 Tavily AI Search 컴포넌트를 도구로 연결할 수 있습니다. Prompt는 에이전트가 질의에 어떻게 답변하고, 응답을 어떤 형식으로 제공하며, 질의와 연구 결과를 플로우 내 다음 에이전트로 어떻게 전달할지 지시합니다.

플로우 내 각 에이전트는 이전 에이전트의 작업을 기반으로 하여, 복잡한 문제 해결을 위한 추론의 연결고리를 만듭니다.

사전 요구사항

  • 실행 중인 Langflow 인스턴스
  • OpenAI API 키
  • Tavily AI API 키

Langflow에서 새 플로우 생성

  1. Langflow를 열고 New Flow를 클릭하세요. Sequential Tasks Agent를 선택합니다. 필요한 컴포넌트가 포함된 스타터 템플릿이 열립니다.

Sequential Tasks Agent 플로우는 다음과 같은 컴포넌트로 구성됩니다.

  • Agent 컴포넌트는 연결된 LLM을 사용해 사용자의 입력을 분석하고, 연결된 도구들 중에서 작업을 수행할 도구를 선택합니다.
  • Chat Input 컴포넌트는 채팅에서 사용자의 입력을 받습니다.
  • Prompt 컴포넌트는 사용자 입력과 사용자가 정의한 프롬프트를 결합합니다.
  • Chat Output 컴포넌트는 플로우의 출력을 채팅에 표시합니다.
  • YFinance 도구 컴포넌트는 Yahoo Finance의 금융 데이터에 접근할 수 있게 해줍니다.
  • Tavily AI Search 도구 컴포넌트는 AI 기반 웹 검색을 수행합니다.
  • Calculator 도구 컴포넌트는 수학 계산을 처리합니다.

Sequential Tasks Agent 플로우 실행 방법

  1. Agent 컴포넌트에 OpenAI API 키를 추가하세요.
  1. Tavily 컴포넌트에 Tavily API 키를 추가하세요.
  1. Playground를 클릭하여 템플릿의 기본 질문으로 채팅 세션을 시작하세요.
Should I invest in Tesla (TSLA) stock right now?
Please analyze the company's current position, market trends,
financial health, and provide a clear investment recommendation.

이 질문은 에이전트에게 어떻게 진행해야 하는지, 어떤 질문에 답해야 하는지에 대해 명확한 지침을 제공합니다.

  1. Playground에서 답변을 확인하며, 에이전트가 Tavily AI Search 도구로 질의를 조사하고, YFinance 도구로 주식 데이터를 분석하며, Calculator를 사용해 해당 주식이 현명한 투자처인지 판단하는 과정을 살펴보세요.
  1. 비슷한 질문을 하면서 에이전트들이 각 도구를 어떻게 활용해 답변하는지 확인해볼 수 있습니다.
