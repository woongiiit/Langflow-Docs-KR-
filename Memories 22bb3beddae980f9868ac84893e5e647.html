Memories

메모리 (Memory) 컴포넌트

: Components > Memories

Langflow의 메모리 (Memory) 컴포넌트는 session_id를 기준으로 채팅 메시지를 저장하고 불러옵니다.

이 컴포넌트는 벡터 스토어 컴포넌트와는 달리, 외부 데이터베이스에 채팅 메시지를 저장하고 검색하기 위해 설계되었습니다.

메모리 컴포넌트는 외부 데이터베이스를 기억장치 (memory)로 사용할 수 있게 함으로써 LLM과 에이전트에게 연속성과 맥락 보존을 가능하게 합니다.

플로우에서 메모리 컴포넌트 사용하기

아래 예시 플로우에서는 Store Message 헬퍼 컴포넌트와 Message History 헬퍼 컴포넌트를 사용하여 Astra DB Chat Memory 컴포넌트로 채팅 기록을 저장하고 불러옵니다.

Store Message 컴포넌트가 채팅 메모리를 Data 객체로 저장하고, Message History 컴포넌트가 저장된 내용을 Data나 string 객체로 불러옵니다.

AstraDBChatMemory 컴포넌트

이 컴포넌트는 AstraDBChatMessageHistory 인스턴스를 생성하여 Astra DB(클라우드 네이티브 데이터베이스 서비스)를 통해 채팅 메시지를 저장하고 검색합니다.

입력값 (Inputs)

이름타입설명
collection_nameString메시지를 저장할 Astra DB 컬렉션 이름 (필수)
tokenSecretStringAstra DB 접근을 위한 인증 토큰 (필수)
api_endpointSecretStringAstra DB 서비스 엔드포인트 URL (필수)
namespaceString컬렉션이 속한 Astra DB 네임스페이스 (선택)
session_idMessageText채팅 세션의 고유 식별자. 미입력 시 현재 세션 ID 사용

출력값 (Outputs)

이름타입설명
message_historyBaseChatMessageHistory해당 세션의 AstraDBChatMessageHistory 인스턴스

CassandraChatMemory 컴포넌트

이 컴포넌트는 CassandraChatMessageHistory 인스턴스를 생성하여 Apache Cassandra 또는 DataStax Astra DB를 통해 채팅 메시지를 저장하고 검색합니다.

입력값 (Inputs)

이름타입설명
database_refMessageTextCassandra contact points 또는 Astra DB 데이터베이스 ID (필수)
usernameMessageTextCassandra 접속 사용자명. Astra DB 사용 시 비워두기
tokenSecretStringCassandra 패스워드 또는 Astra DB 토큰 (필수)
keyspaceMessageTextCassandra keyspace 또는 Astra DB 네임스페이스 (필수)
table_nameMessageText저장할 테이블 또는 컬렉션 이름 (필수)
session_idMessageText채팅 세션의 고유 식별자 (선택)
cluster_kwargsDictionaryCassandra 클러스터 추가 설정을 위한 키워드 인수 (선택)

출력값 (Outputs)

이름타입설명
message_historyBaseChatMessageHistory해당 세션의 CassandraChatMessageHistory 인스턴스

Mem0 Chat Memory 컴포넌트

Mem0 Chat Memory 컴포넌트는 Mem0 메모리 스토리지를 사용하여 채팅 메시지를 저장하고 검색합니다.

입력값 (Inputs)

이름표시 이름설명
mem0_configMem0 ConfigurationMem0 메모리 인스턴스 초기화에 사용할 설정 사전
ingest_messageMessage to Ingest메모리에 인제스트할 메시지 내용
existing_memoryExisting Memory Instance기존 Mem0 메모리 인스턴스 (선택)
user_idUser ID메시지와 연관된 사용자 식별자
search_querySearch Query관련 메모리를 검색하기 위한 입력 텍스트
mem0_api_keyMem0 API KeyMem0 플랫폼 API 키. 로컬 버전 사용 시 비워두기
metadataMetadata인제스트되는 메시지에 추가할 메타데이터
openai_api_keyOpenAI API KeyOpenAI 임베딩 사용 시 필요한 API 키 (컴포넌트 설정에 없으면 필수)

출력값 (Outputs)

이름표시 이름설명
memoryMem0 Memory인제스트 후 생성된 Mem0 메모리 객체
search_resultsSearch Results메모리 검색 결과

ZepChatMemory 컴포넌트

이 컴포넌트는 ZepChatMessageHistory 인스턴스를 생성하며, Zep 메모리 서버를 통해 채팅 메시지를 저장하고 검색합니다.

입력값 (Inputs)

이름타입설명
urlMessageTextZep 인스턴스 URL (필수)
api_keySecretString인증을 위한 API 키 (필수)
api_base_pathDropdown사용할 API 버전 (예: api/v1, api/v2)
session_idMessageText채팅 세션의 고유 식별자 (선택)

출력값 (Outputs)

이름타입설명
message_historyBaseChatMessageHistory해당 세션의 ZepChatMessageHistory 인스턴스
