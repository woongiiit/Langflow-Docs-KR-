Agents

에이전트(Agent) 컴포넌트

: Components > Agents

1. 에이전트 컴포넌트란?

Langflow에서 에이전트(Agent) 컴포넌트는 플로우 안에서 인공지능 에이전트의 동작 방식과 기능을 정의합니다.

이 컴포넌트를 이용하면, 다양한 도구(툴) 컴포넌트들과 연동해 실제로 문제를 해결하거나 질문에 답하는 자동화된 AI 에이전트를 만들 수 있습니다.

2. 에이전트와 LLM, 그리고 Tool(툴)의 역할

에이전트는 **LLM(대형 언어 모델, 예: OpenAI GPT 등)**을 두뇌처럼 사용해, 현재 상황에 가장 적합한 도구를 스스로 선택합니다.

  • Tool(툴)이란?

    에이전트가 외부 정보를 얻거나 특정 작업을 실행할 수 있게 해주는 기능입니다.

    예를 들어, 계산기, URL 정보 추출, 벡터DB 검색 등이 모두 ‘툴’입니다.

  • 각 툴은 Tool 객체로 등록되어, 에이전트가 어떤 일을 할 수 있는지 설명(description)이 붙어 있습니다.
  • 에이전트는 시작 시점에 ‘사용 가능한 툴 목록’을 받아, 그 중 필요한 기능을 골라 사용하게 됩니다.

에이전트가 문제를 해결할 때, 연결된 LLM이 여러 툴 중에서 어떤 것을 써야 할지 논리적으로 판단하게 됩니다.

3. 플로우에서 에이전트 사용 예시

Langflow의 기본 예제 플로우(Simple agent)에서는 에이전트 컴포넌트가 URL 툴, 계산기 툴 등과 연결되어 사용자의 질문에 답을 해줍니다.

OpenAI와 같은 LLM이 에이전트의 두뇌 역할을 하며, 각 툴은 에이전트 컴포넌트의 Tools 포트에 연결됩니다.

※ 여러 에이전트를 활용한 예시는 ‘에이전트로 플로우 만들기’(Create a flow with an agent) 문서를 참고하세요.

4. 에이전트 컴포넌트 주요 특징

이 컴포넌트를 사용하면 아래와 같은 설정이 가능합니다:

  • LLM 모델 연동: 원하는 LLM(예: OpenAI 등)을 연결해 사용할 수 있습니다.
  • 시스템 메시지 프롬프트: 에이전트의 기본 행동 지침이나 초기 맥락을 설정할 수 있습니다.
  • Tools 포트: 다양한 툴 컴포넌트를 연결해 에이전트의 기능을 확장할 수 있습니다.

자세한 정보는 별도의 Agent 컴포넌트 문서에서 확인하세요.

5. 에이전트 컴포넌트 파라미터

입력값(Inputs)

이름타입설명
agent_llmDropdown에이전트가 사용할 LLM 제공자 선택 (예: OpenAI, 기타, Custom)
system_promptString에이전트의 동작 지침 및 초기 맥락 제공
toolsList에이전트가 사용할 수 있는 툴 목록 (비워둘 수도 있음)
input_valueString에이전트가 처리할 작업이나 질문
add_current_date_toolBoolean활성화 시, 현재 날짜를 반환하는 툴 추가
memoryMemory대화 히스토리 등 상태 저장용 (선택적)
max_iterationsInteger에이전트가 반복 실행 가능한 최대 횟수
handle_parsing_errorsBoolean실행 중 파싱 에러를 처리할지 여부
verboseBoolean상세 로그를 남길지 여부

출력값(Outputs)

이름타입설명
responseMessage입력된 작업/질문에 대한 에이전트의 답변

6. 레거시(지원 종료) 에이전트 컴포넌트

과거 버전의 에이전트 컴포넌트들도 사용할 수 있지만, 더 이상 공식 지원되지 않습니다.

(1) JSON Agent

  • 설명: JSON 또는 YAML 파일과 LLM을 활용해 JSON 기반 에이전트를 생성하는 컴포넌트
  • 입력값
    • llm: 사용할 언어 모델
    • path: JSON 또는 YAML 파일 경로
  • 출력값
    • agent: 생성된 JSON 에이전트 인스턴스

(2) Vector Store Agent

  • 설명: LangChain을 활용해 벡터스토어 기반 에이전트를 생성
  • 입력값
    • llm: 사용할 언어 모델
    • vectorstore: 벡터스토어 정보
  • 출력값
    • agent: 벡터스토어 에이전트 인스턴스

(3) Vector Store Router Agent

  • 설명: 여러 벡터스토어를 라우팅하는 에이전트 생성
  • 입력값
    • llm: 사용할 언어 모델
    • vectorstores: 벡터스토어 정보 리스트
  • 출력값
    • agent: 벡터스토어 라우터 에이전트 인스턴스

*이 문서는 https://docs.langflow.org/components-agents 문서를 번역/편집한 것으로 생략되거나 추가된 내용이 있습니다. 자세한 사항은 원문을 참고하시기 바랍니다.
