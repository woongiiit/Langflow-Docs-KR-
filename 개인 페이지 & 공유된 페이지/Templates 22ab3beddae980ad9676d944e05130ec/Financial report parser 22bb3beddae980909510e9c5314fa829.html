Financial report parser

LLM 응답을 구조화된 형식으로 파싱하기 위해 Structured output 및 Parser 컴포넌트를 활용한 Financial Report Parser 플로우를 만듭니다.

이 예시에서는 Chat Input 컴포넌트에 샘플 재무 보고서가 미리 입력되어 있어, 매출총이익(Gross Profit), EBITDA, 순이익(Net Income) 추출 과정을 시연합니다.

사전 요구사항

  • 실행 중인 Langflow 인스턴스
  • OpenAI API 키

Financial Report Parser 플로우 생성 방법

  1. Langflow 대시보드에서 New Flow를 클릭합니다.
  1. Financial Report Parser를 선택합니다.

Financial Report Parser 플로우가 생성됩니다.

메모리 챗봇 플로우 실행

  1. OpenAI 모델에 OpenAI API 키를 추가하세요.
  1. Playground 버튼을 클릭한 다음 Send를 클릭하세요. Chat Input 컴포넌트에는 시연을 위해 샘플 재무 보고서가 미리 입력되어 있습니다. 채팅은 구조화된 응답을 반환합니다.
EBITDA: $900 million , Net Income: $500 million , GROSS_PROFIT: $1.2 billion

이 정보가 어떻게 추출되었는지 확인하려면 플로우를 살펴보세요.

  1. Structured Output 컴포넌트에서 출력 스키마 테이블을 확인하려면 Open table을 클릭하세요. Structured Output 컴포넌트는 연결된 OpenAI 모델 컴포넌트를 "두뇌"로 사용하여, 정의된 스키마에 따라 재무 데이터를 DataFrame으로 추출합니다.
| Name         | Description           | Type | Multiple |
|--------------|-----------------------|------|----------|
| EBITDA       | description of field  | text | False    |
| NET_INCOME   | description of field  | text | False    |
| GROSS_PROFIT | description of field  | text | False    |
  1. 추출된 데이터가 포함된 템플릿을 확인하려면, Parser 컴포넌트의 Template 필드에서 Scan 아이콘을 클릭하세요. Parser 컴포넌트는 추출된 데이터를 채팅에서 사용할 수 있도록 형식화된 메시지로 변환합니다. 각 변수는 구조화된 출력에서 값을 받아옵니다.
EBITDA: {EBITDA}  ,  Net Income: {NET_INCOME} , GROSS_PROFIT: {GROSS_PROFIT}
