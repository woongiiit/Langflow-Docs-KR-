플로우 발행하기

Langflow는 플로우를 외부 애플리케이션에 발행하고 통합하는 여러 방법을 제공합니다. API 엔드포인트(endpoint, 접근 가능한 통신 종점)로 플로우를 노출하거나, 웹사이트에 채팅 위젯으로 임베드하거나, 공개 Playground로 공유하는 등 사용자가 플로우에 접근할 수 있도록 만드는 다양한 옵션을 이 가이드에서 다룹니다.

API 접근

API 접근 패널은 플로우를 외부 애플리케이션에 통합하기 위한 코드 템플릿을 제공합니다.

  • Python
    • Python 탭에는 Python requests 라이브러리를 사용하여 사용자의 플로우(flow)와 상호작용하는 코드가 표시됩니다.
    1. 이 코드를 복사하여 파이썬 스크립트 파일(예: python-test-script.py)에 붙여넣으세요.
    1. 스크립트를 실행하세요.
      1. python3 python-test-script.py --message="tell me about something interesting"
  • JavaScript
    • JavaScript API 탭은 JavaScript에서 플로우와 상호작용하는 코드를 표시합니다.
    1. JavaScript 파일에 코드를 복사하여 붙여넣기 하세요.
    1. 스크립트를 실행하세요.node test-script.js "tell me about something interesting"
  • curl
    • cURL 탭에는 사용자의 플로우(flow)에 쿼리(query)를 전송하기 위한 샘플 코드가 표시됩니다.
    • 이 코드를 복사하여 실행하면 플로우에 쿼리를 전송하고 그 결과를 받을 수 있습니다.

응답 내용은 플로우에 따라 달라집니다. 엔드포인트가 성공적인 응답을 반환하는지 확인하세요.

임시 오버라이드(Temporary overrides)

임시 오버라이드 탭은 플로우에 사용 가능한 매개변수들을 표시합니다. 매개변수를 수정하면 모든 창에서 코드 매개변수가 변경됩니다. 예를 들어, 채팅 입력 컴포넌트의 input_value를 변경하면 이 플로우의 /run 엔드포인트에 대한 모든 API 호출에서 해당 값이 변경됩니다.

API로 파일을 플로우에 보내기

Langflow API에 파일을 보내는 방법에 대한 자세한 내용은 파일 엔드포인트를 참조하세요.

사이트에 임베드하기

사이트에 임베드하기 탭은 HTML의 <body> 태그 안에 삽입하여 플로우와 상호작용할 수 있는 코드를 표시합니다.

자세한 내용은 임베디드 채팅 위젯을 참조하세요.

공유 가능한 Playground

공유 가능한 Playground는 Langflow 애플리케이션의 Playground를 /public_flow/$FLOW_ID 엔드포인트에 노출시킵니다.

Ngrok이나 zrok과 같은 공유 플랫폼을 사용하여 이 엔드포인트를 공개적으로 공유할 수 있습니다.

Datastax Langflow를 사용하는 경우, 조직(Organization) 내의 모든 사용자와 URL을 공유할 수 있습니다.
