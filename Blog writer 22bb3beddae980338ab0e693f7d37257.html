Blog writer

OpenAI를 사용하여 one-shot 애플리케이션용 블로그 작성기 플로우를 구축하세요.

이 flow은 기본 프롬프팅 흐름을 URL 및 Parser 컴포넌트로 확장하여 여러 URL에서 콘텐츠를 가져오고 로드된 데이터를 일반 텍스트로 변환합니다.

OpenAI는 Text Input 및 Prompt 컴포넌트의 지시에 따라 이 로드된 데이터를 사용하여 블로그 게시물을 생성합니다.

사전 요구사항

  • 실행 중인 Langflow 인스턴스
  • OpenAI API 키

블로그 작성기 흐름 만들기

  1. Langflow 대시보드에서 New Flow를 클릭하세요.
  1. Blog Writer를 선택하세요.
  1. Blog Writer 플로우가 생성됩니다.

이 플로우는 Prompt, OpenAI 및 Chat Output 컴포넌트로 블로그 글 생성기를 만들고, URL과 Text Input 컴포넌트에서 참조 콘텐츠와 지시사항으로 증강합니다.

URL 컴포넌트는 하나 이상의 웹 링크에서 원시 텍스트와 메타데이터를 추출합니다. Parser 컴포넌트는 URL 컴포넌트에서 오는 데이터를 프롬프트에 입력할 일반 텍스트로 변환합니다.

플로우의 프롬프트를 검토하려면, Prompt 컴포넌트의 Template 필드를 클릭하세요.

블로그 작성기 플로우 실행하기

  1. Playground 버튼을 클릭한 다음 Run Flow를 클릭하세요. Langflow에 관한 블로그 게시물이 생성되며, 콘텐츠는 langflow.org와 docs.langflow.org에서 가져옵니다.
  1. 다른 주제에 대해 작성하려면 URL 컴포넌트의 값을 변경하고 Playground의 왼쪽 사이드바에서 Instruction을 조정하세요. 다시 시도하여 LLM이 어떤 내용을 구성하는지 확인해보세요.
