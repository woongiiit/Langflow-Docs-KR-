Prompts

프롬프트 (Prompt) 컴포넌트

: Components > Prompts

프롬프트(Prompt)는 대형 언어 모델(LLM)에 사용자 입력과 변수를 처리하는 방법을 지시하는 구조화된 입력입니다.

프롬프트 컴포넌트는 사용자 정의 필드와 동적 변수를 사용하여 모델에 구조화된, 반복 가능한 프롬프트 템플릿을 생성합니다.

프롬프트는 자연어와 중괄호로 만든 변수를 조합한 형태입니다.

플로우에서 Prompt 컴포넌트 사용하기

프롬프트를 수정하는 예시는 Vector RAG 스타터 플로우에서 찾을 수 있습니다. 이 플로우에서는 기본 챗봇 플로우에 전체 벡터 RAG 파이프라인을 확장한 모습을 보여줍니다.

Prompt 컴포넌트의 기본 프롬프트는

Answer the user as if you were a GenAI expert, enthusiastic about helping them get started building something fresh.

입니다.

이 프롬프트는 LLM의 채팅 상호작용에 특정 성격(personality)을 부여하지만, 템플릿화할 때 유용한 변수는 포함되어 있지 않습니다.

프롬프트 템플릿을 수정하려면 Prompt 컴포넌트에서 Template 필드를 클릭하세요. 예를 들어, {context} 변수는 LLM이 임베딩된 벡터 데이터를 활용해 더 나은 답변을 생성할 수 있도록 합니다.

Given the context
{context}
Answer the question
{user_question}

변수를 프롬프트 템플릿에 추가하면 컴포넌트에 새로운 필드가 자동으로 생성됩니다. 이러한 필드는 다른 컴포넌트로부터 텍스트 입력을 받아 프롬프트 자동화에 사용하거나, 다른 컴포넌트로 지침을 출력하는 데 연결할 수 있습니다. 순차 작업 에이전트(starter flow)에서 프롬프트를 사용해 에이전트의 동작을 제어하는 예시를 확인할 수 있습니다.

파라미터

입력값 (Inputs)

이름표시 이름 (Display Name)설명
templateTemplate동적 변수를 포함한 프롬프트 템플릿을 생성합니다.

출력값 (Outputs)

이름표시 이름 (Display Name)설명
promptPrompt Messagebuild_prompt 메서드가 반환하는 생성된 프롬프트 메시지입니다.

Langchain Hub 프롬프트 템플릿

중요: 이 컴포넌트는 Components 메뉴의 Bundles 아래에서 사용할 수 있습니다.

이 컴포넌트는 Langchain Hub에서 프롬프트를 가져옵니다. 프롬프트가 로드되면 사용자 정의 변수를 위한 입력 필드가 자동으로 생성됩니다. 예를 들어, 기본 프롬프트 "efriis/my-first-prompt"는 profession과 question 필드를 생성합니다.

파라미터

입력값 (Inputs)

이름표시 이름 (Display Name)설명
langchain_api_keyYour LangChain API Key사용할 LangChain API 키입니다.
langchain_hub_promptLangChain Hub Prompt사용할 LangChain Hub 프롬프트입니다.

출력값 (Outputs)

이름표시 이름 (Display Name)설명
promptBuild Promptbuild_prompt 메서드가 반환하는 생성된 프롬프트 메시지입니다.

*이 문서는 https://docs.langflow.org/components-prompts 문서를 번역/편집한 것으로 생략되거나 추가된 내용이 있습니다. 자세한 사항은 원문을 참고하시기 바랍니다.
