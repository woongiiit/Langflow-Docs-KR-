LangFlow 에이전트 사용하기

에이전트는 LLM(대규모 언어 모델)을 두뇌로 활용하여 자율적으로 문제를 분석하고 해결에 필요한 도구를 선택합니다.

LangFlow의 에이전트 컴포넌트는 에이전트 구성을 간소화하여 애플리케이션 개발에 집중할 수 있게 해줍니다.

에이전트 컴포넌트는 다양한 LLM 제공업체와 사용자 지정 지침을 포함하여 에이전트를 만드는 데 필요한 모든 것을 제공합니다.

에이전트 설정

에이전트 컴포넌트를 원하는 제공업체와 모델, 사용자 지정 지침 및 도구를 사용하도록 구성할 수 있습니다.

에이전트 모델 및 제공업체

모델 제공업체(Model Provider)와 모델 이름(Model Name) 설정을 사용하여 에이전트가 사용할 LLM을 선택합니다.

OpenAI 모델을 위한 OpenAI API 키와 같이 선택한 제공업체에 대한 인증 키를 제공해야 합니다.

에이전트 지침 및 입력

에이전트 지침(Agent Instructions) 필드에서 에이전트 컴포넌트가 모든 대화에 사용하도록 할 사용자 지정 지침을 제공할 수 있습니다.

이러한 지침은 런타임에 제공되는 입력(Input)에 추가로 적용됩니다.

에이전트 도구

에이전트는 요청을 완료하는 데 적절한 도구를 사용할 수 있을 때 가장 유용합니다.

에이전트 컴포넌트에 연결하기만 하면 어떤 LangFlow 컴포넌트든 도구로 사용할 수 있습니다.

컴포넌트를 도구로 연결할 때는 도구 모드(Tool Mode)를 활성화하여 해당 컴포넌트를 도구로 구성해야 합니다.

자세한 내용은 에이전트용 도구 구성을 참조하세요.

플로우에서 에이전트 컴포넌트 사용하기

LangFlow에서 에이전트 컴포넌트부터 시작하여 바깥쪽으로 작업하며 에이전트를 만듭니다.

  1. 새 플로우(New Flow)를 클릭한 다음 빈 플로우(Blank Flow)를 클릭합니다.
  1. 에이전트 컴포넌트를 작업 공간에 추가합니다.
  1. 기본 모델을 사용하거나 다른 제공업체와 모델을 선택한 다음 선택한 제공업체에 대한 자격 증명을 제공합니다. 예를 들어, 기본 모델을 사용하려면 OpenAI API 키를 제공해야 합니다.
  1. 채팅 입력(Chat input) 및 채팅 출력(Chat output) 컴포넌트를 플로우에 추가하고 도구 호출 에이전트에 연결합니다.

이 기본 플로우를 사용하면 Playground에서 에이전트와 채팅할 수 있지만, 단지 OpenAI LLM과 채팅하는 것에 불과합니다. 에이전트 컴포넌트의 강력한 기능을 활용하려면 몇 가지 도구를 연결하세요.

  1. 뉴스 검색(News Search), URL, 계산기(Calculator) 컴포넌트를 플로우에 추가합니다.
  1. 뉴스 검색, URL, 계산기 컴포넌트에서 도구 모드를 활성화합니다. 컴포넌트의 헤더 메뉴에서 도구 모드를 활성화하여 컴포넌트를 에이전트와 함께 사용할 수 있게 합니다.

도구 모드는 컴포넌트의 입력을 수정하여 에이전트 컴포넌트가 도구의 사용 가능한 작업을 사용하도록 요청을 수락할 수 있도록 함으로써 컴포넌트를 도구로 만듭니다. 도구 모드의 컴포넌트에는 도구 세트(Toolset) 포트가 있으며, 에이전트가 도구의 작업을 사용하도록 하려면 이 포트를 에이전트 컴포넌트의 도구(Tools) 포트에 연결해야 합니다.

  1. 세 도구 컴포넌트의 도구 세트 포트를 에이전트 컴포넌트의 도구 포트에 연결합니다.
  1. Playground를 엽니다. 에이전트에게 내 질문에 답하기 위해 어떤 도구를 사용하고 있나요?라고 물어보세요. 에이전트는 연결된 도구 목록으로 응답해야 합니다.

에이전트의 예시 응답:

저는 내장 지식(2024년 6월까지)과 외부 도구 세트를 조합하여 질문에 답변합니다. 제가 사용할 수 있는 주요 도구 유형은 다음과 같습니다:
웹 검색 및 콘텐츠 가져오기: 웹 페이지에서 콘텐츠를 가져와 요약할 수 있으며, 링크를 재귀적으로 크롤링할 수도 있습니다.
뉴스 검색: RSS 피드를 통해 Google 뉴스를 사용하여 최신 뉴스 기사를 검색할 수 있습니다.
계산기: 산술 계산을 수행하고 수학적 표현식을 평가할 수 있습니다.
날짜 및 시간: 다양한 시간대에서 현재 날짜와 시간을 제공할 수 있습니다.
이러한 도구는 최신 정보 제공, 계산 수행, 필요할 때 인터넷에서 특정 데이터 검색 등에 도움이 됩니다. 특정 질문이 있으시면 알려주시고, 가장 적절한 도구를 사용하여 도와드리겠습니다!
  1. 에이전트에게 오늘의 기술 뉴스를 요약해 주세요라고 요청하세요. Playground는 에이전트의 도구 호출, 제공된 입력, 그리고 에이전트가 요약을 생성하기 전에 받은 원시 출력을 표시합니다. 에이전트는 뉴스 검색 컴포넌트의 search_news 작업을 호출해야 합니다.

이제 LangFlow 에이전트로 플로우를 성공적으로 구성했습니다. 더 전문화된 문제를 해결하기 위해 더 많은 도구를 연결하세요.
