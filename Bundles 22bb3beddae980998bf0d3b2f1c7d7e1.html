Bundles

번들 (Bundles) 컴포넌트

: Components > Bundles

번들 (Bundles)은 외부 서비스 제공자(프로바이더)별로 묶여 제공되는 서드파티 컴포넌트 모음입니다.

자세한 내용은 각 번들 제공자의 공식 문서를 참고하시기 바랍니다.

아래는 Bundles 메뉴에 포함된 대표적인 에이전트(Agent) 컴포넌트들입니다.

1. CrewAI Agent

CrewAI Agent는 역할(role), 목표(goal), 배경(backstory), 사용할 도구(tools) 등을 설정하여 특정 임무를 수행하는 AI 에이전트를 만들 수 있는 컴포넌트입니다. 여러 에이전트를 구성한 ‘크루’를 만들어 협업 형태 작업도 가능합니다.

입력값 (Inputs)표기설명
roleRole에이전트의 역할
goalGoal에이전트의 목표
backstoryBackstory에이전트의 배경
toolsTools에이전트가 사용할 수 있는 도구 목록
llmLanguage Model에이전트가 사용하는 언어 모델
memoryMemory에이전트가 메모리를 사용할지 여부
verboseVerbose상세 로그 출력 여부
allow_delegationAllow Delegation다른 에이전트에게 작업 위임 허용 여부
allow_code_executionAllow Code Execution코드 실행 허용 여부
kwargskwargs에이전트에 전달할 추가 키워드 인자
출력값 (Outputs)표기설명
outputAgent생성된 CrewAI 에이전트 객체

자세한 내용은 CrewAI 공식 문서를 참고하세요.

2. Hierarchical Crew

이 컴포넌트는 여러 에이전트를 계층 구조로 조직해 매니저가 관리하는 크루를 구성하고, 여러 작업(Task)을 분배해 협업하도록 합니다.

입력값 (Inputs)표기설명
agentsAgents크루 구성원 에이전트 객체 리스트
tasksTasks실행할 HierarchicalTask 객체 리스트
manager_llmManager LLM매니저 에이전트용 언어 모델
manager_agentManager Agent매니저 역할을 할 특정 에이전트
verboseVerbose상세 로그 출력 여부
memoryMemory크루의 메모리 설정
use_cacheUse Cache결과 캐싱 사용 여부
max_rpmMax RPM분당 최대 요청 횟수
share_crewShare Crew크루 정보를 에이전트 간 공유할지 여부
function_calling_llmFunction Calling LLM함수 호출용 언어 모델
NameDisplay NameInfo
crewCrew계층형 작업 실행이 가능한 Crew 객체

자세한 설명은 CrewAI 문서를 참고하세요.

3. CSV Agent

CSV 파일과 LLM을 기반으로 표 형식 데이터 작업이 가능한 에이전트를 만들 수 있습니다.

입력값 (Inputs)표기설명
llmLanguageModel사용할 언어 모델
pathFileCSV 파일 경로
agent_typeAgent Type생성할 에이전트 유형
출력값 (Outputs)표기설명
agentAgentExecutor생성된 CSV 에이전트 인스턴스

4. OpenAI Tools Agent

OpenAI LLM과 다양한 도구(Tools)를 연동하여 문제 해결형 에이전트를 구성하는 컴포넌트입니다.

입력값 (Inputs)표기설명
llmLanguageModel사용할 언어 모델
toolsList of Tools에이전트에 제공할 툴 목록
system_promptSystem Prompt에이전트의 맥락 제공용 시스템 프롬프트
input_valueInput Value에이전트가 처리할 사용자 입력
memoryMemory맥락 유지를 위한 메모리
max_iterationsInteger에이전트가 반복 실행할 최대 횟수
verboseBoolean중간 단계 출력 등 상세 로그 출력 여부
handle_parsing_errorsBoolean실행 중 파싱 에러를 처리할지 여부
출력값 (Outputs)표기설명
agentAgentExecutorOpenAI Tools 에이전트 인스턴스
outputString입력 실행에 대한 결과 출력

5. OpenAPI Agent

OpenAPI 명세를 기반으로 외부 API와 자동 상호작용하는 에이전트를 구성할 수 있는 컴포넌트입니다.

입력값 (Inputs)표기설명
llmLanguageModel사용할 언어 모델
openapi_specOpenAPI Spec서비스의 OpenAPI 명세
base_urlBase URLAPI의 기본 URL
headersDictAPI 요청용 선택적 헤더
agent_executor_kwargsAgent Executor kwargs에이전트 실행에 필요한 추가 인자
출력값 (Outputs)표기설명
agentAgentExecutorOpenAPI 에이전트 인스턴스

tasksTasks실행할 SequentialTask 객체 리스트
verboseVerbose상세 로그 출력 여부

use_cacheUse Cache결과 캐싱 사용 여부
max_rpmMax RPM분당 최대 요청 횟수
share_crewShare Crew크루 정보를 에이전트 간 공유할지 여부
function_calling_llmFunction Calling LLM함수 호출용 언어 모델
출력값 (Outputs)표기설명
crewCrew순차 작업 실행 Crew 객체

7. Sequential Task Agent

CrewAI 기반의 에이전트를 이용해 **순차 작업 구조(Task + Agent)**를 정의할 수 있는 컴포넌트입니다.

입력값 (Inputs)표기설명
roleRole에이전트의 역할
goalGoal에이전트의 목표
backstoryBackstory에이전트의 배경 이야기
toolsTools에이전트가 사용할 수 있는 도구 목록
llmLanguage Model에이전트가 사용하는 언어 모델
memoryMemory에이전트가 메모리를 사용할지 여부
verboseVerbose상세 로그 출력 여부
allow_delegationAllow Delegation다른 에이전트에게 작업 위임 허용 여부
allow_code_executionAllow Code Execution코드 실행 허용 여부
agent_kwargsAgent kwargs추가 키워드 인자
task_descriptionTask Description작업의 목적과 실행에 대한 설명
expected_outputExpected Task Output기대되는 작업 결과
async_executionAsync Execution비동기 작업 실행 여부
previous_taskPrevious Task이전 작업(연결용)
출력값 (Outputs)표기설명
task_outputSequential Task생성된 SequentialTask 객체 리스트

8. SQL Agent

SQL 데이터베이스와 연동하여 쿼리 작성/실행 및 결과 해석을 수행하는 에이전트 목적 컴포넌트입니다.

입력값 (Inputs)표기설명
llmLanguageModel사용할 언어 모델
databaseDatabaseSQL 데이터베이스 연결
top_kIntegerSELECT 쿼리 반환 결과 개수
use_toolsBoolean쿼리 실행에 툴 사용 여부
return_intermediate_stepsBoolean중간 단계 반환 여부
max_iterationsInteger최대 반복 횟수
max_execution_timeInteger최대 실행 시간(초)
early_stopping_methodString조기 종료 방식
verboseBoolean에이전트의 중간 생각 출력 여부
출력값 (Outputs)표기설명
agentAgentExecutorSQL 에이전트 인스턴스

return_intermediate_stepsBoolean중간 단계 반환 여부
max_iterationsInteger최대 반복 횟수
max_execution_timeInteger최대 실행 시간(초)
early_stopping_methodString조기 종료 방식

출력값 (Outputs)표기설명

입력값 (Inputs)표기설명

출력값 (Outputs)표기설명
