플로우(Flows)

Langflow에서 플로우는 완전히 저장 가능한 형태로 변환될 수 있으며 파일 시스템에서 저장하고 불러올 수 있습니다. 이 가이드에서는 플로우를 가져오고 내보내는 방법을 살펴보겠습니다.

플로우 가져오기(Import Flow)

이미 Langflow JSON 파일이 있다면, 프로젝트 이름을 클릭하고 Import Flow(플로우 가져오기)를 선택하여 Langflow로 가져올 수 있습니다.

가져오기가 완료되면 바로 플로우를 사용할 수 있습니다.

플로우 내보내기(Export Flow)

플로우를 내보내는 옵션은 위에서 보여진 메뉴와 동일한 곳에서 사용할 수 있습니다. JSON으로 내보내면 다른 Langflow 인스턴스(다른 설치 환경)로 플로우를 가져올 수 있습니다.

Langflow JSON 파일 내용

Langflow JSON 파일에는 컴포넌트와 연결을 설명하는 노드(nodes)와 엣지(edges), 그리고 플로우를 설명하는 추가 메타데이터가 포함되어 있습니다.

Langflow JSON 파일의 예시는 Langflow 저장소에 있는 Basic Prompting.json 파일을 살펴보세요.

노드(Nodes)

노드는 플로우를 구성하는 컴포넌트(구성 요소)를 나타냅니다.

ChatInput 노드는 플로우의 시작점입니다. 이것이 가장 먼저 실행되는 노드입니다.

ChatInput-jFwUm은 노드의 고유 식별자입니다.

아래는 노드의 JSON 구조 예시입니다(일부 줄바꿈 제거함):

{
  "data": {
    "description": "Get chat inputs from the Playground.",
    "display_name": "Chat Input",
    "id": "ChatInput-jFwUm",
    "node": {
      "base_classes": ["Message"],
      "description": "Get chat inputs from the Playground.",
      "display_name": "Chat Input",
      "icon": "MessagesSquare",
      "template": {
        "input_value": {
          "display_name": "Text",
          "info": "Message to be passed as input.",
          "value": "Hello"
        },
        "sender": {
          "value": "User",
          "options": ["Machine", "User"]
        },
        "sender_name": {
          "value": "User"
        },
        "should_store_message": {
          "value": true
        }
      }
    },
    "type": "ChatInput"
  },
  "position": {
    "x": 689.5720422421635,
    "y": 765.15583413140
  }
}

엣지(Edges)

엣지는 노드 간의 연결을 나타냅니다.

ChatInput 노드와 OpenAIModel 노드 사이의 연결은 다음과 같은 엣지로 표현됩니다:

{
  "className": "",
  "data": {
    "sourceHandle": {
      "dataType": "ChatInput",
      "id": "ChatInput-jFwUm",
      "name": "message",
      "output_types": ["Message"]
    },
    "targetHandle": {
      "fieldName": "input_value",
      "id": "OpenAIModel-OcXkl",
      "inputTypes": ["Message"],
      "type": "str"
    }
  },
  "id": "reactflow__edge-ChatInput-jFwUm{œdataTypeœ:œChatInputœ,œidœ:œChatInput-jFwUmœ,œnameœ:œmessageœ,œoutput_typesœ:[œMessageœ]}-OpenAIModel-OcXkl{œfieldNameœ:œinput_valueœ,œidœ:œOpenAIModel-OcXklœ,œinputTypesœ:[œMessageœ],œtypeœ:œstrœ}",
  "source": "ChatInput-jFwUm",
  "sourceHandle": "{œdataTypeœ: œChatInputœ, œidœ: œChatInput-jFwUmœ, œnameœ: œmessageœ, œoutput_typesœ: [œMessageœ]}",
  "target": "OpenAIModel-OcXkl",
  "targetHandle": "{œfieldNameœ: œinput_valueœ, œidœ: œOpenAIModel-OcXklœ, œinputTypesœ: [œMessageœ], œtypeœ: œstrœ}"
}

이 엣지는 ChatInput 컴포넌트가 Message 타입을 target 노드(여기서는 OpenAIModel 노드)로 출력함을 보여줍니다. OpenAIModel 컴포넌트는 input_value 필드에서 Message 타입을 입력받습니다.

추가 메타데이터 및 프로젝트 정보

플로우에 대한 추가 정보는 루트(최상위) data 객체에 저장됩니다.

  • 메타데이터 및 프로젝트 정보: 플로우의 이름, 설명, last_tested_version(마지막으로 테스트된 버전) 등을 포함합니다.
{
  "name": "Basic Prompting",
  "description": "Perform basic prompting with an OpenAI model.",
  "tags": ["chatbots"],
  "id": "1511c230-d446-43a7-bfc3-539e69ce05b8",
  "last_tested_version": "1.0.19.post2",
  "gradient": "2",
  "icon": "Braces"
}
  • 시각적 정보: 작업 공간에서 플로우의 초기 위치를 정의하는 정보입니다.
"viewport": {
  "x": -37.61270157375441,
  "y": -155.91266341888854,
  "zoom": 0.7575251406952855
}

노트(Notes)는 작업 공간 내에서 플로우를 이해하는 데 도움이 되는 주석과 같습니다. 링크, 코드 스니펫(코드 조각) 및 기타 정보를 포함할 수 있습니다. 노트는 마크다운으로 작성되며 node 객체로 저장됩니다.

{
  "id": "undefined-kVLkG",
  "node": {
    "description": "## 📖 README\nPerform basic prompting with an OpenAI model.\n\n#### Quick Start\n- Add your **OpenAI API key** to the **OpenAI Model**\n- Open the **Playground** to chat with your bot.\n..."
  }
}
